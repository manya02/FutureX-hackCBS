About

SV Converter

We are building a system that will convert your story into video format. For example, suppose you're telling a friend about an incident. What if I told you that you can see the entire incident on a real-time video?

It is a machine learning project that uses natural language processing concepts to process our input data and then uses the DALL.E.2 concept to generate images from pre-processed texts. The images are then combined using Python's opencv library to create a video story.
